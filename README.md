CMS Platform

A simple, modern CMS built with Next.js, Express, and MongoDB. It’s designed to be lightweight, secure, and flexible — something that just works without unnecessary complexity.

Overview

This CMS lets you manage content efficiently through a clean dashboard, backed by an Express API and MongoDB database.
Authentication is handled using OAuth (Google, Auth0, or GitHub). It supports multiple roles like admin, editor, and viewer, and can be extended for more complex use cases.
